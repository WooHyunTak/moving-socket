## Moving Service Web Socket Server
- 이사매칭 플랫폼의 실시간 채팅을 위한 소켓 서버

- 로그인된 고객과 기사의 실시간 소통을 통하여 서비스 질의 향상을 목표로 한다.
- 상태방의 서비스 로그인상태를 확인하여 사용자로 하여금 상대방을 확인 하도록 한다.
